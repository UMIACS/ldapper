Installation
======================

Via Python Package
------------------

Install the latest version from PyPI:

.. code-block:: console

    pip install ldapper


Via Source
----------

The project is hosted at https://github.com/umiacs/ldapper and can be installed
using git:

.. code-block:: console

    git clone https://github.com/umiacs/ldapper.git
    cd ldapper
    python setup.py install
